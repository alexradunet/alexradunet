## Project positioning

### Nazar

Nazar is a local-first personal AI environment: a private, sovereign agent layer for memory, tools, workflows, and the user’s own machine. It explores what personal computing looks like when an AI agent is a first-class part of the operating environment rather than a cloud chatbot.

### forge.dance

forge.dance is a gamified street dance learning app where mastered moves become collectible cards that generate personalized daily workouts. It combines movement practice, progression systems, and mobile-first product design.
